# Multimodal Biometric Authentication System Using Deep Learning

## Project Overview

This project is a secure authentication system that verifies user identity using multiple biometric traits including Face Recognition, Fingerprint Verification, and Signature Verification. The system stores biometric records of registered users and provides authentication only when the input matches the stored data.

This multimodal approach improves security, reduces fraud, and increases authentication accuracy compared to single biometric systems.

---

## Features

- Face Recognition Authentication
- Fingerprint Verification
- Signature Verification
- User Registration and Record Storage
- Secure Authentication Process
- Rejection of Unauthorized Users
- Improved Accuracy using Deep Learning Models
