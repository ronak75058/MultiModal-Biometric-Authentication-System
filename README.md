MultiModal-Biometric-Authentication-System

A secure Multi-Modal Biometric Authentication System built using Python, OpenCV, and machine learning techniques for intelligent and reliable user authentication. The system uses biometric verification methods such as face recognition to improve security and authentication accuracy.

Features
Face registration and authentication
Real-time face detection using webcam
Multi-modal biometric verification
User registration system
Secure authentication process
Easy-to-use interface
Computer vision-based identity validation

Installation
1. Clone the Repository
git clone https://github.com/YOUR_USERNAME/MultiModal-Biometric-Authentication-System.git
cd MultiModal-Biometric-Authentication-System
2. Create Virtual Environment
python -m venv venv

Activate virtual environment:

Windows
venv\Scripts\activate
Linux / Mac
source venv/bin/activate
3. Install Dependencies
pip install -r requirements.txt
Run the Project
python main.py
